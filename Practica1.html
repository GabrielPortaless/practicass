¿Que es Html?


Es un lenguaje de marcado de gtexto que sirve para presentar paginas web desde el punto de vista del front end;un front end en español significa la parte final que un usuario de internet vera.Una etiqueta (marca)delimita cada no de los elementos que compone un documento de Html.El cuerpo de yba pagina web las paginas web estan formadas por 2 partes el encabezado y el cuerpo de la pagina, la primera tiene la función de mostrar configuraciones y el cuerpo es donde se mostrara en s toda la pgina.La etiqueta html es el inicio de una pagina web en cualquier vercion pero existe una etiqueta llamada doctype html que incia una pagina web con la vercion 5.
